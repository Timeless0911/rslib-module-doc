# Rslib project

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get started

Build the library:

```bash
pnpm build
```

Build the library in watch mode:

```bash
pnpm dev
```

Start the doc dev server:

```bash
pnpm dev:doc
```

Build the doc for production:

```bash
pnpm build:doc
```

Preview the production build locally:

```bash
pnpm preview:doc
```
